# Hierarchical 2D Transformations

An interactive computer graphics demonstration illustrating hierarchical transformations using matrix algebra and the HTML5 Canvas API.

Users can manipulate joint rotations with sliders to explore how local transformations combine to produce articulated motion.

---

## Features

- Hierarchical coordinate systems
- Matrix transformations
- Translation
- Rotation
- Scaling
- Interactive joint controls
- Real-time rendering

---

## Technologies

- JavaScript
- HTML5 Canvas
- glMatrix
- Linear Algebra

---

## Concepts Demonstrated

- Transformation matrices
- Local vs world coordinate systems
- Matrix multiplication
- Hierarchical modeling
- Articulated animation

---

## Controls

Four sliders independently rotate different joints in the articulated model, demonstrating how transformations propagate through a hierarchy.

---

## Running

Clone the repository and open

index.html

in your browser.

No build process required.
